Playing with Images

Basic Image with alt and title attributes

The MDN web docs has a great introduction to Responsive images. Please read that for additional tips.

The basic syntax for adding an image is to use the img element. It's also important to add an alt attribute that describes the image for screen readers. The title attribute is optional but can be used to show a title of the image when a mouse cursor hovers over.

Changing the width and height

We can use the height and width attributes to resize an image. However, it's better to change the height and width in a photo editor, like Photoshop or Gimp, instead. This is because despite the change here, the original file may be large, and it's better to avoid transmitting large files over the internet when possible. We don't always know what kind of internet connection people may have. Keep files as small as possible.

Responsive design with image

Using the above methods to set the size of an image is not altogether responsive, either. In order to make an image responsive to different screen sizes, we can use the srcset and sizes HTML attributes to list optional size images to display depending on screen size. This allows the browser to load different size images depending on the size of the screen. This has other bandwidth implications. Some people may be on mobile devices, and therefore may have slower internet connections. Using a smaller image file will cause the smaller image to load first, and therefore, faster on such devices. The src attribute is used here as a fallback device, in case srcset does not work.

I only list two sizes here, but more could be added to adjust for different file and screen sizes.

Background Images

Background images can be applied over the background of a particular part of a page, like this section here, or to the entire page using the body elcodeent.

The code for something like that would look like the code below. Note I use url and not src to set the file location. The url here is called a CSS reference function, unlike src, which is a HTML attribute. We only use src for specific HTML elements like the img, audio, video elements.

In the example below, I placed file in the bottom right corner, but you can do more, including repeating an image or placing it in different locations. See: CSS3 Background Image.

  
    section {
    background-image: url(images/littlesquare.png);
    background-repeat: no-repeat;
    background-position: right bottom;
    }
